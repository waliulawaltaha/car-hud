# car-hud
Car HUD (Heads-Up Display) is a high-performance, distraction-free telemetry system designed specifically for the Even Realities G2 smart glasses. Unlike traditional dashboard displays that require you to look down, this application uses a Peripheral AR Layout to beam critical engine data directly into your field of vision, keeping the center of your view clear for the road.

Key Features:
* Peripheral HUD Design:
  * Top Left: Digital Speedometer (toggles between MPH and KM/H).
  * Top Right: Smart Gear Indicator (calculated based on velocity).
  * Bottom Edge: A dynamic, horizontal RPM Gauge that visualizes engine load without cluttering your line of sight.

* Intelligent Alert System: Features a high-visibility "!!! SHIFT UP !!!" center-screen warning that flashes only when the RPM exceeds a safe threshold (3500+ RPM).

* Material Expressive Control Center: A sleek, dark-mode mobile interface that allows you to pair your car, toggle units, and monitor connection status with a single tap.
